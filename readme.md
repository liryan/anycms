### 通用CMS管理系统，基于Laravel框架+PHP+MySQL开发

#下载

`git clone https://gitee.com/canbetter/anycms`

#初始化安装

1.运行　composer dump

2.修改.env文件中的DB_HOST,DB_USER,DB_PASSWORD配置数据库

3.运行　./artisan install ./database/admlite.sql 倒入数据表,此处的dbname和.env中要保持一直
